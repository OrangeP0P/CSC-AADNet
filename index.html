This project page presents an overview of CSC-AADNet, our solution submitted to the ICASSP 2026 EEG Auditory Attention Decoding Challenge. The goal of this work is to achieve robust attention decoding from electroencephalography (EEG) under multi-modal conditions.

In the challenge, we introduce a cross-calibrated network designed to mitigate spatio-temporal distribution shifts. Our method leverages unsupervised covariance alignment and lightweight normalization to handle the data heterogeneity. To further enhance robustness, we incorporate stability-aware training combined with ensemble inference. As demonstrated in our experiments, CSC-AADNet effectively improves generalization across both different subjects and recording sessions.

Method Overview


    (A) Cross-Calibrated Block: Tackles cross-subject and cross-session data shift via subject-level calibration, temporal smoothing, and channel-wise normalization. (B) Stability-Aware Training: Enhances robustness by optimizing for stable predictions during training iterations. (C) Model Ensemble: Aggregates predictions from top-k models to reduce variance and improve accuracy on unseen subjects.

Usage Guide

1 Quick Start: Training

Download the ICASSP 2026 dataset and configure the data path in trainer_all_ensemble.py. You can also customize the preprocessing pipeline in the getData function (e.g., normalization, smoothing).

trainer_all_ensemble.py 
# 1. Set your dataset path
DATA_PATH = 'your_local_path/to/dataset/'

# 2. Configure preprocessing (Default settings)
seq_alldata_ao, alllabel_ao = getData(
    args, 
    audio_only_sub_ids,
    channel_normalization=True,   # Z-score normalization
    smoothing=True,               # Temporal smoothing
    ma_window=3                   # Moving average window size
)

3 Ensemble Inference

To reproduce our submission results, gather the checkpoints from the selected folds into a directory. Update inference.py to point to these models and your test data.

inference.py 
# Path to the directory containing selected .pt files
args.model_dir = "/path/to/your/selected_models/" 

# Path to the test dataset
args.data_path = "/path/to/test_data/"

# Run inference
# python inference.py
